# customer-management-system
This program is a personal project which serves as a customer management system. It is desktop application primarily written using the Delphi programming language as well as Python for invoice geneeration.

The program is able to keep track of customer details and payments made. From there invoices may be generated as well as statements, however, the statements feature has not been implemented yet.

All customers are stored on a Microsoft Access database. For invoice generation an executable file (compiled from python) is run to generate these invoices using the openpyxl library. An adaption to this code as a standalone is avaliable as well in my repositories as invoice-management-system.

More in-depth details of how the program works will be published at the completetion of the statements feature.
